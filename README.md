# 🚗 自动驾驶政策追踪器

> 本项目由 **小马佩德罗** 自动追踪中国自动驾驶相关政策，每日自动更新。

## 📋 数据来源

- **交通运输部** (MOT) - 智能交通、自动驾驶运输服务规范
- **工信部** (MIIT) - 智能网联汽车准入、试点政策
- **公安部** (MPS) - 道路测试管理、事故处理
- **国务院 & 发改委** - 顶层战略规划

## 🔄 自动更新

- **触发时间**: 每日 11:00 (北京时间)
- **更新内容**: 最新政策、动态、法规变化
- **托管方式**: GitHub Pages

## 📁 项目结构

```
av-policy-tracker/

```
